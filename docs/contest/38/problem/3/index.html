C. 不等关系

题目描述

给定一个字符串 ，仅包含 < 和 > 两种字符。

你需要计算「使得 当且仅当 为 < 的排列 」的数量。

可以发现，答案可能很大，因此你只要输出它对 取模的结果。

输入格式

从标准输入读入数据。

输入一行一个由 < 和 > 组成的字符串 。

输出格式

输出到标准输出。

输出一行一个整数，表示满足要求的排列数量对 取模的结果。

样例

样例输入 1

<><>>

样例输出 1

35

样例说明 1

举例来说，排列 是一个的满足要求的排列。

而排列 不是一个的满足要求的排列，因为它不满足 。

样例输入 2

<><<>>><><<><>>

样例输出 2

497133532
